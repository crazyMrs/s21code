SmartCalc v3.0 - Справка

Добро пожаловать в SmartCalc v3.0!

Интерфейс программы включает следующие элементы:

1. Поле ввода - здесь вы можете вводить математические выражения для вычисления.
2. Метка "Input x:" и поле для ввода x - здесь вы можете ввести значение переменной x.
3. Кнопка "Calculate" - нажмите эту кнопку для выполнения вычислений.
4. Поле истории - здесь отображаются предыдущие вычисления.
5. Кнопка для выбора выражения из истории.
6. Кнопка для построения графика.
7. Поля для ввода диапазон значений для графика.

Программа поддерживает основные арифметические операции и функции.
